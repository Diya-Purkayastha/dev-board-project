Question-1: What are the different ways to select an element in the DOM?
There are many different ways like using id(getElementById) , class(getElementsByClassName), tagname(getElementsByTagName), queryselectors(querySelector,querySelectorAll). For selecting multiple element use tagname or class or querySelectorAll.We often use the id and querySelectors.
Question-2: What is the difference between innerHTML, innerText, and textContent ?
  • innerHTML-- gets or sets the HTML content of an element,can be used to insert or modify HTML elements dynamically.
  • innerText-- gets or sets the visible text content of an element,can be used to insert or modify text elements dynamically, ignores hidden elements.
  • innerContent-- gets or sets the text content of an element,include all child node, does not interpret HTML.
    • innerContent is more better in performance than innerHTML or innerText
    Question-3: What is event delegation in the DOM?
    Event delegation is a technique in JS where a single event listener is added to a parent element instead of multiple listeners for each child element. The event listener utilizes event bubbling to handle events on dynamically added or existing child elements.
    Using event.target, can identify the specific child that fired the event.
    Question-4: What is event bubbling in the DOM?
    Event bubbling is a concept in DOM where an event starts from the target element and propagates up through its parent elements until it reaches the document object.
    Question-5: How do you create, add, and remove elements using JavaScript?
  • Creating Elements: use document.createElement() to create a new HTML element. After creating an element developer can easily set element by using innerHTML or innerText or innerContent
  • Adding Elements: After creating element, one can use
  • appendChild() -- adds as last child
  • append() -- can add multiple elements and text
  • prepend() -- adds as first child
  • Removing Elements: After Adding element, one can use those method to remove
  • remove() -- removes an element directly
  • removeChild() -- removes a child from a parent
